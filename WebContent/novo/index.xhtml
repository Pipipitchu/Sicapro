SICAPRO

Este projeto foi desenvolvido pelos Alunos: Jeferson Euclides e Rodrigo Adriano, para obtenção de nota parcial na discliplina de Laboratório de Programação VI, disciplina esta ministrada pelo Prof. Me. Roberto Ribeiro Rocha. O SICAPRO (Sistema Integrado de Cadastro e Apoio a Projeto) é um sistema web que visa auxiliar no gerenciamento de projeto, apresentando opções de cadastro, atualizações e deleções e relatórios conforme solicitado pelo cliente. O código do projeto está disponível pelo github: Rodrigo / Jeferson
